## ✨ Features

- ⚡ **30-second invoices** — fill in client, items, and amount — done
- 📄 **Instant PDF generation** — download or print directly from your browser
- 💼 **Professional templates** — clean, client-ready design that builds trust
- ➕ **Line item support** — add unlimited services or products with quantities and rates
- 🔢 **Auto-calculations** — subtotal, tax, and total computed in real time
- 🏷️ **Custom branding** — add your business name, logo, and payment details
- 🔒 **Zero data leakage** — everything stays local, nothing is stored or transmitted

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/invoicekit.git
cd invoicekit
npm install
npm run dev
```

Or use the [live version](https://invoicekit.vercel.app) — no setup needed.
